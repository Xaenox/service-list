# Community Services Marketplace

## Что это?

Внутренний маркетплейс услуг для членов клуба. Платформа, где участники сообщества могут:
- **Предлагать** свои товары и услуги другим членам клуба
- **Находить** нужные услуги среди проверенных людей из сообщества

## Для кого?

### Поставщики услуг
Предприниматели, фрилансеры и владельцы бизнесов из клуба, которые хотят:
- Найти клиентов среди "своих"
- Предложить особые условия для членов сообщества
- Получить доверие за счёт принадлежности к клубу

### Потребители услуг
Члены клуба, которые:
- Ищут исполнителей и поставщиков среди знакомого круга
- Хотят получить бонусы и скидки как члены сообщества
- Предпочитают работать с проверенными людьми

## Ключевые возможности

### Каталог услуг
- Название и описание услуги
- Категории (IT, Юридические услуги, Маркетинг и т.д.)
- Формат работы: онлайн, офлайн или гибрид
- Геолокация для офлайн-услуг

### Профиль поставщика
- Имя и аватар из клуба
- Контакты: email, телефон, Telegram, WhatsApp, сайт
- Город и страна

### Бонусы для членов клуба
Каждый поставщик может указать специальное предложение:
- Скидка (% или фиксированная сумма)
- Бесплатный пробный период
- Подарок при заказе
- Любое другое специальное условие

### Поиск и фильтрация
- Текстовый поиск по названию и описанию
- Фильтр по категориям
- Фильтр по формату (онлайн/офлайн)
- Фильтр по городу
- Показать только с бонусами

## Роли пользователей

| Роль | Возможности |
|------|-------------|
| **Член клуба** | Просмотр каталога, создание своих услуг, редактирование своих услуг |
| **Модератор** | + удаление любых услуг (нарушающих правила) |
| **Администратор** | + управление категориями |

## Доступ

Авторизация только через аккаунт клуба. Доступ имеют только активные члены с оплаченной подпиской.

## Примеры использования

**Сценарий 1: Поиск разработчика**
> Мне нужен iOS разработчик. Открываю каталог → фильтр "IT" → фильтр "Онлайн" → нахожу 5 предложений от членов клуба со скидками 10-20%.

**Сценарий 2: Размещение услуги**
> Я юрист, хочу найти клиентов в клубе. Создаю услугу "Юридическое сопровождение бизнеса" → указываю контакты → добавляю бонус "Первая консультация бесплатно".

**Сценарий 3: Локальный поиск**
> Ищу бухгалтера в Москве для личных встреч. Фильтр "Офлайн" → город "Москва" → категория "Финансы".
